# easySocket_set_non_block(\_)

Seat a socket to non-blocking for a given id.

## **Declaration**

```C
void easySocket_set_non_block(
                        int sock_id
                    )
```

## **Parameters**
**sock_id**  
The id to set the socket to non-blocking.


## **Result**
Nothing is returned.
